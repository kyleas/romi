\doxysubsection{Detailed Description}
Provides high-\/level kinematic calculations for the ROMI robot. 

The \doxylink{class_romi}{Romi} class calculates the robot\textquotesingle{}s linear and angular motion (x, y, heading) using left/right wheel velocities. It contains methods to calculate velocity, reset distances, and handle heading offsets.
