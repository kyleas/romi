\doxysubsection{Detailed Description}
Provides an interface to the BNO055 \doxylink{class_i_m_u}{IMU} for orientation and calibration. 

Communicates with the BNO055 sensor over I2C to read euler angles, gyroscope, magnetometer data, and handle calibration. Can store/read calibration data from a binary file to speed up repeated usage.
