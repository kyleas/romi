\doxysubsection{Detailed Description}
\doxylink{class_motor}{Motor} driver interface for the ROMI chassis. 

This module provides a class to control a DC motor via PWM signals and a direction pin. It works with drivers like the DRV8838 for Pololu\textquotesingle{}s ROMI chassis, featuring functions to set motor effort, enable, and disable the driver.
