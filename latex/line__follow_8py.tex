\doxysubsection{Detailed Description}
Implements a line-\/following strategy using \doxylink{class_i_r}{IR} sensors. 

Contains a class that takes \doxylink{class_i_r}{IR} sensor data and outputs left/right wheel velocities to follow a line. The user can set a maximum velocity, and this module will scale the left/right speeds based on the sensor centroid.
