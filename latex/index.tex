\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Overview}{Overview}}\label{index_autotoc_md1}
Welcome to the {\bfseries{\doxylink{class_romi}{Romi} Time-\/\+Trials Project\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}—a self-\/navigating robot designed for autonomous competition in \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+ME 405\+: Mechatronics}}. ~\newline
 The goal of this project is to build a {\bfseries{highly precise, efficient, and autonomous \doxylink{class_romi}{Romi} robot}} that can\+:
\begin{DoxyItemize}
\item {\bfseries{Follow a predefined track}} using calibrated infrared (\doxylink{class_i_r}{IR}) sensors.
\item {\bfseries{Adjust its heading}} using an Inertial Measurement Unit (\doxylink{class_i_m_u}{IMU}).
\item {\bfseries{Utilize \doxylink{class_p_i_d}{PID} control}} for precise motor velocity regulation.
\item {\bfseries{Respond to obstacles}} with bumper sensors.
\end{DoxyItemize}

This project integrates real-\/time task scheduling, control theory, and embedded systems to create a functional and adaptable robotic platform.

\hypertarget{index_autotoc_md5}{}\doxysection{\texorpdfstring{System Architecture}{System Architecture}}\label{index_autotoc_md5}
The system is built on a task-\/based multitasking model, where each component operates as a separate task\+:




\begin{DoxyEnumerate}
\item {\bfseries{Task 1}}\+: Handles user inputs and data logging.
\item {\bfseries{Task 2 \& Task 3}}\+: Control left and right motors using \doxylink{class_p_i_d}{PID} velocity control.
\end{DoxyEnumerate}




\begin{DoxyEnumerate}
\item {\bfseries{Task 4}}\+: Implements path following using \doxylink{class_i_r}{IR} sensor and \doxylink{class_i_m_u}{IMU} data.
\end{DoxyEnumerate}




\begin{DoxyEnumerate}
\item {\bfseries{Task 5}} (Optional)\+: Provides an emergency stop using bumper sensors.
\end{DoxyEnumerate}
