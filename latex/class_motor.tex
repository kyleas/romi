\doxysubsection{Detailed Description}
A motor driver interface encapsulated in a Python class. 

This class creates a motor driver object controlling PWM and direction pins for a single motor. The driver uses separate PWM and direction lines, e.\+g., DRV8838 modules on a ROMI chassis.
